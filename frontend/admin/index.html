Platform Documentation

Platform documentation lives in two places- the Google doc, "Digital Platform Beta Documentation" and the wiki for the "digital-platform-beta" project in Github. The Google doc contains ALL of the documentation, but any documentation that can be public-facing is in the Github wiki and gets updated there instead of in the Google doc version. The internal-only pieces of the documentation are updated in the Google doc.

Editing entries in the database via the BOP Digital Platform

On each Admin page, you can use the icon next to an item to edit that entry in the database. With the exception of the Sites page, which has a form editor, entries in the database are written in JavaScript Object Notation, so only edit those entries if you understand the syntax.

Here are links and descriptions of the main Admin pages:

  • DOE Schools List- Every NYC Public School (whether they participate or not) and its economic and demographic data (pulled from NYC Open Data) and any BOP-related information we have for that school. Loads VERY slowly.
  • Events- All events, past and present.
  • Lessons- All lessons, whether their status is published, draft, returned, or pending.
  • Organizations- All organizations who have signed up for the platform + their ORS sites + expeditions (if any), + economic and demographic data for NYC public schools.
  • People- All participants and their org sites (if any) + whether they have attended basic training, + info about their organization
  • Sites- All sites + NYC Parks info + permit info

Permits

The "Permits" page is designed to generate the full NYC Department of Environmental Conservation License to Collect and Possess (LCP) permit report for Oyster Research Stations. The current code generates the report for the 2017-2018 season. You will need to edit the code by changing the dates in the binding at the top of the permits.html file for the 2018-2019 season, but that should be the only change you need to make.
